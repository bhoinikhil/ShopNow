# 🛒 ShopNow.com – React E-commerce Website

ShopNow.com is a modern **E-commerce Web Application** built using **ReactJS and Bootstrap**.  
The application allows users to browse products, view product details, add items to the cart, and manage shopping easily.

This project demonstrates **React component architecture, routing, state management, and responsive UI design**.

---

# 🚀 Tech Stack

- ReactJS
- HTML5
- CSS3
- Bootstrap
- JavaScript (ES6)

---

# ✨ Features

- Responsive E-commerce UI
- Product Listing
- Category Based Products (Men / Women / Kids)
- Product Details Page
- Shopping Cart
- Add to Cart functionality
- Seller Product Upload
- Discount Calculation
- Quantity Management
- Navbar with Search
- Footer with Links
